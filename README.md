# Yantra Theme Converter
A tool to convert old-style theme posts for Yantra Launcher to the newer JSON configurations.

# Demo
https://github.com/user-attachments/assets/2a83e476-9980-4feb-ba37-a54f35e46f29

# Usage
1. Copy the message text from Discord of the theme you would like to convert.
1. Go to https://eeriemyxi.github.io/yantra-themeconv/
1. Paste the text in the text area then hit "Convert."
1. Your result should have been copied to your clipboard.
